# Specification

## Summary
**Goal:** Build a complete digital platform for Rabindra Bharati Art Academy with public pages, secure admission system, role-based student/teacher portals, live streaming classes, and art-specific teaching features.

**Planned changes:**
- Create artistic, mobile-friendly public homepage with academy information, location, contact details, and WhatsApp chat button
- Build public admission and gallery pages displaying academy information and best student artwork
- Implement online admission form with student/guardian details, artwork upload, and teacher approval workflow
- Create secure login system with separate student and teacher roles (no public signup)
- Build student dashboard showing profile, current exam level, messages, homework with reference images, and embedded live class stream
- Implement 11-level exam progression system (Adhya → Madhya → Purna → 1st-8th Year) with one exam per year and teacher-controlled promotion
- Build comprehensive teacher panel to manage admissions, students, level assignments, promotions, and view progress analytics
- Create messaging and homework system for level-specific notices, assignments, reference images, and exam instructions
- Implement art teaching features including reference paintings, sketch demos, color guides, and practice topics
- Build high-quality live streaming system embedded in dashboard (teacher broadcasts, students watch)
- Create gallery management for teachers to upload artwork, tag by level, and set public/private visibility
- Build teacher analytics dashboard with total students, students per level, growth tracking, and charts
- Implement secure backend with role-based access, password encryption, and protected student data storage

**User-visible outcome:** A complete art academy platform where visitors can browse information and apply for admission, students can log in to view their profile and level-specific content including homework and live classes, and teachers can manage the entire academy through an admin panel with student management, content delivery, and analytics.
